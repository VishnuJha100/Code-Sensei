# 🚀 Code Sensei - AI-Powered Code Review

Welcome to **Code Sensei**, an intelligent code review platform powered by Google Gemini API! 🔥  
This website analyzes your code, provides insights, and suggests improvements based on industry best practices.

## 🌟 Features:
✅ AI-Powered Code Review  
✅ Quality Rating (Better, Good, Normal, Bad)  
✅ Suggestions for Improvement  
✅ Step-by-Step Code Explanation  
✅ Bug Detection & Fix Recommendations  
✅ Supports Multiple Programming Languages  

## 🛠️ Tech Stack:
- **Frontend:** React + Vite, Monaco Editor
- **AI Integration:** Google Gemini API  
- **Styling:** Tailwind CSS  
- **State Management:** React Hooks  

## 🚀 Installation:
1. Clone the repository:
   ```sh
   git clone https://github.com/VishnuJha100/Code-Sensei.git
   cd Code-Sensei
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Add environment variables in .env (root directory)
   ```sh
   VITE_API_KEY=your_google_gemini_api_key
   ```
4. Start the development server
   ```sh
   npm run dev
   ```

## 🔧 Usage:
- Enter your code in the Monaco Editor
- Select your programming language
- Click Review Code to analyze the code using Google Gemini API
- View feedback & suggestions in real-time

## 📦 API Integration
This project utilizes Google Gemini API for AI-driven code analysis.
